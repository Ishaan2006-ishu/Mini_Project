# MockMate Pro

AI Powered Mock Interview System

## Overview
MockMate Pro is an AI-powered platform that helps users practice technical interviews with real-time feedback.

## Features
- Role-based interview questions
- AI answer evaluation
- Interview performance tracking
- User dashboard

## Tech Stack
Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express

Database:
- MongoDB

AI:
- Gemini / OpenAI API

## Installation

Clone the repository

git clone https://github.com/Ishaan2006-ishu/Mini_Project.git

Install dependencies

npm install

Run the server

node server/server.js
